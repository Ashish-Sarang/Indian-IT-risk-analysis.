# Indian IT Risk Analysis

This project compares the short-term performance and risk of TCS,Infosys,Wipro and HCLTech using Python.

The analysis uses daily stock price data to calculate:

- Total Return
- Daily Volatility
- 10-Day Volatility
- 95% Value at Risk (VaR)

The project was created to practice financial risk analysis and Python-based data analysis while studying FRM concepts.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

## Files

- `project risk analysis.ipynb` – Main analysis notebook
- CSV files containing stock price data
